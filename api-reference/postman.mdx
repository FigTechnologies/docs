## How to Import

1. Download the JSON file above.
2. Open Postman and click the **Import** button in the top left.
3. Drag and drop the downloaded file into the import window.
4. Set up your environment variables (API Key, Base URL) and you're ready to go!

<Tip>
  Don't forget to configure your `api_key` in the collection variables or environment.
</Tip>
